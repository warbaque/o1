CarMap 

class CarMap extends MapPanel

A "car map" is the graphical map shown in a CarSim application window. It may contain any number of cars which are drawn onto the map. A car map listens for mouse commands that signal the creation of cars or the execution of car methods.

NOTE TO STUDENTS: In this course, you don't need to understand how this class works or can be used.
