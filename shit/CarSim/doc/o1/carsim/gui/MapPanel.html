MapPanel 

Companion object MapPanel

class MapPanel extends Component with Publisher with TileLoaderListener

The class MapPanel is a Scala wrapper around the JMapViewer class of the JMapViewer Java library. It provides a zoomable world map.

NOTE TO STUDENTS: In this course, you don't need to understand how this class works or can be used.
