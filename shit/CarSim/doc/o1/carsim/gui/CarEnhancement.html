CarEnhancement 

class CarEnhancement extends AnyRef

The class CarEnhancement represents cars using the student-created Car class to do it. Its functionality contains some things we didn't want to put in the car exercise.

NOTE TO STUDENTS: In this course, you don't need to understand how this class works or can be used.

(This class is not really a part of the GUI per se, but is here as an artifact of the programming assigment.)
